spacerJS
========

## About plugin
space key appearance custom for fixed header site

## Demo site
[spacerJS](http://spacerjs.anticyb.org/)

## usage
- read jQuery
  ``<script src="http://code.jquery.com/jquery-1.10.1.min.js"></script>``
- read spacerJS
  ``<script src="js/spacer.min.js></script>"``
- set
  ``$('.headerNav').spacer();``
