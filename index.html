固定ヘッダーだと思ってくれ

spacerJS

ヘッダー固定とかにしてるサイトでスペースキーを押した時に文字が隠れないようにするjQueryプラグイン

Build by Grunt written CoffeeScript.

download spacerJS

about

スペースキーでページをスクロールしてる感じのオールドタイプな人はヘッダーを固定表示にしてるサイトが嫌いなはず。僕も嫌い。だってスペースを押してスクロールした際に、固定されてるヘッダーに文字が被ってちゃんと読めなくなるもん。

だからそういう人、つまりオールドタイプ向けのサイトを作る際にちょっと一手間付け加えてあげて「お、このサイトは分かってるな」感を提供したいわけです。つかしてあげて。やってるとこはやってる。

この一手間に対応してくれてるサイトがあまりないので悔しくなってプラグインにしました。試しに使ってみてはいかがですかね。これが流行ると僕が喜びます。

usage

  1. jQuery本体を読み込む 
    <script src='http://code.jquery.com/jquery-1.10.1.min.js'></script>
  2. spacerJSを読み込む 
    <script src='js/spacer.min.js'></script>
  3. 固定にしてるヘッダーをセレクタにして実行 
    $('.headerNav').spacer();
  4. オプション指定でオフセット量とか変えられる 
    $('.headerNav').spacer({offset:100});
